Blackjack

Developed front-end and backend website using HTML, CSS, Javascript, React.js. The site is simple in appearance but is also accessible for individuals who rely on text to speech to play having each element identified. Users are given an authentic game of blackjack where all the table rules apply. The algorithm used keeps the game interesting and non repetitive.
